青少年Simper xor

发表于2023-03-24|更新于2023-04-22
|阅读量:

这是青少年ctf上的一个题，下面是我对这个图的分析

接下来就是脚本咯：

#include<stdio.h>
int main()
{
    char v3[]="a7shw9o10e63nfi19dk";
    char compare_date[]={'G','d','\x18','3','\x10','a','Q',';','5','^','c','d','\x1D','t','\x19','M','`','\x1B','i'};
    char v7[19];
    char buffer1[18];
    char buffer2[19];
    char flag[19]="Huan-y1n5";//先根据第一个循环求前9个
    for(int k=1;k<=18;k++)
    {
        v3[k] ^= flag[(k - 1) % 9] ^ flag[(k - 1) % 7] ^ v3[k - 1];
    }
    for(int i=0;i<=18;i++)
    {
        v7[i]=compare_date[i]^v3[i];
    }
    buffer2[0]='I';
    for(int i=0;i<18;i++)//这里求出最开始v7的前8个
    {
        buffer2[i+1]=buffer2[i]^v7[i]^0x53;
    }
    for (int i = 1; i < 19; ++i )
    {
        flag[i]=buffer2[i]^buffer2[i-1];
    }
    for (int i = 0; i < 19; ++i )
    {
        printf("%c",flag[i]);
    }
 return 0;
}

在这个题目中要注意的是先根据v3求出flag的前九个字符，然后写脚本。
